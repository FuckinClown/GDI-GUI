
# GDI-GUI

A visuals creator for fun. No payload is harmful, its all for show

I might add a few features or abandon this after its done, if you like this project please star the repo so I know what you think



## Support

This uses the g++ compiler to create a dynamic GDI effect base on what you selected

if you do not have a g++ compiler installed, it will error.

I recomend watching [this video](https://www.youtube.com/watch?v=8CNRX1Bk5sY) on how to install MinGW, however Msys64 will also work.
